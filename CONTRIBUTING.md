# Guia de Contribuição

Obrigado por considerar contribuir com este projeto!

## Como contribuir
1. Abra uma issue para discutir melhorias ou reportar bugs.
2. Faça um fork do repositório e crie uma branch para sua contribuição.
3. Envie um pull request detalhando suas alterações.

## Padrão de Código
- Siga as boas práticas de React e JavaScript.
- Utilize nomes de variáveis e funções claros.
- Mantenha o código limpo e documentado.

## Revisão
Todos os pull requests passarão por revisão antes de serem aceitos.

## Código de Conduta
Ao contribuir, você concorda em seguir o [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
